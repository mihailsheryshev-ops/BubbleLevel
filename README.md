# BubbleLevel — Пузырьковый уровень

Мобильное приложение для Android, реализующее цифровой пузырьковый уровень с использованием акселерометра.

## Описание
Приложение считывает данные акселерометра и отображает угол наклона устройства в виде линейного пузырькового уровня. При горизонтальном положении пузырёк окрашивается в зелёный цвет и появляется надпись «Ровно!».

## Требования
- Android 7.0 и выше (API 24+)
- Устройство с акселерометром

## Запуск
1. Скачай и установи Android Studio
2. Открой проект: File → Open → выбери папку проекта
3. Подключи Android-устройство по USB
4. Нажми Run ▶

## Стек технологий
- Kotlin
- Android SDK
- SensorManager / Accelerometer
- Canvas API
